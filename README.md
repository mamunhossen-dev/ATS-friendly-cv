# ATS-Friendly CV Generator

A modern, responsive ATS-Friendly CV/Resume Generator built with **HTML, CSS, and Vanilla JavaScript**.
Create professional resumes with **live preview**, **PDF export**, **ATS score analysis**, and **keyword matching**—all in a single-page web app.

---

## Features

* **Live Resume Preview** – Instantly see your CV update as you type
* **ATS-Friendly Formatting** – Clean single-column structure optimized for Applicant Tracking Systems
* **PDF Export** – Download resume as PDF
* **Print Support** – Print-ready A4 layout
* **ATS Score Checker** – Analyze CV completeness and get suggestions
* **Keyword Match Checker** – Compare CV against job descriptions
* **Local Storage Support** – Auto-save data in browser
* **Responsive Design** – Works on desktop and mobile
* **Dynamic Sections** – Add/remove experience, education, skills, certifications, and projects
